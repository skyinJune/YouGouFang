# YouGouFang
### 一、下载安装运行
#### 1> clone
```
git clone git@github.com:skyinJune/YouGouFang.git
```
#### 2> 安装依赖
<b>分别cd到client和server目录下npm i</b>

#### 3> 跑起来跑起来！
<b>cd 到client下</b>
```
npm start
```
